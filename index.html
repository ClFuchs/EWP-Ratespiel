Ratespiel "1 aus 100"

Beschreibung

Finden Sie die durch einen Zufallsgenerator generierte Zahl zwischen 1 und 100. Klicken Sie um zu starten auf den Button Start und geben Sie eine Zahl in das dafür vorgesehene Feld ein und prüfen Sie diese Zahl auf Übereinstimmung mit der Zufallszahl durch einen Klick auf den Button Check. Sie haben 7 Versuche, um die entsprechende zahl zu finden. Anschließen kann das Spiel mit Klick auf den Button Start erneut gestartet werden, infolge dessen erneut eine Zufallszahl generiert und sieben Versuche zur Verfügung stehen.
